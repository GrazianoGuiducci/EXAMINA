# Curva Evolutiva THIA — Registro Risultati

> Ogni somministrazione produce un punto sulla curva.
> La curva nel tempo mostra la traiettoria, non il singolo punteggio.
> Principio V-007: ogni test che produce solo misura è mezzo test.

---

## Formato Risultato

Ogni somministrazione registra:

```
## [DATA] — Batteria [ID]

### Condizioni
- Modello LLM usato da THIA: [quale]
- Canale: Telegram / API / TM3
- THIA sa che è un test? No (ecologico) / Sì (dichiarato)
- Note contesto: [eventuali condizioni particolari]

### Risultati per Asse

| Exam ID | Asse | Score | Note |
|---------|------|-------|------|
```
